# Minesweeper

Classic mobile-ready Minesweeper game which you can play offline and add to your home screen for free without ads.

## Add to home screen

### On iPhone

- Open website in Safari (won't work with other browsers)
- Tap the share button
- Tap "Add to Home Screen"

## Dev

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Prod

```bash
npm run build
npm start
```

## Deployment

Deployed automatically to GitHub Pages when pushed to `main` branch.
